# Mirror UB

Proyecto estatico listo para subir a Vercel.

## Archivos

- `index.html`: pantalla del carnet.
- `espacio.html`: pantalla del espacio personal.
- `styles.css`: estilos moviles.
- `vercel.json`: configuracion de rutas para Vercel.

## Despliegue en Vercel

1. Sube esta carpeta a un repositorio de GitHub.
2. En Vercel, crea un proyecto nuevo desde ese repositorio.
3. Deja la configuracion por defecto:
   - Framework Preset: `Other`
   - Build Command: vacio
   - Output Directory: vacio
4. Deploy.

La pagina principal sera `/` y la segunda pantalla estara en `/espacio`.
